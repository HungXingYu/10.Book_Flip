<h2 id="1"> 專案概述--Project Description</h2>

> 想要感謝作者：木樨，而嘗試製作的 Side Project
>
> 功能僅有：
>
> 1.  可右翻閱讀木樨作者的《奇蹟》相關所有二創作品
> 2.  Menu 可選擇系列作品
> 3.  目錄連結可直接翻頁到對應頁面
> 4.  輸入頁碼後按下 Enter 鍵可翻頁至指定頁碼
>
> 文章版權皆木樨所有，若閱讀之後有任何感想請多多回饋給木樨。
